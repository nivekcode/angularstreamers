## How to Contribute

To get started with the project on your local machine:

1. **Clone the Repository**:
   ```bash
   git clone <repository-url>
   ```
2. **Navigate to the Project Directory**:
   ```bash
   cd <project-directory>
   ```
3. **Install Dependencies**:
   ```bash
   npm install
   ```
4. **Start the Project**:
   ```bash
   npm start
   ```
